再担保业务授信的报告

申报单位：风险管理部
保密级别：内部公开
报告编号：SURETY-2025-089
日期：2024年11月24日

一、 综合授信情况概述

基于 2024 年度全省政府性融资担保体系运行数据及 2025 年宏观经济预测，本年度拟对 48 家合作机构进行再担保业务授信。总规模预计达到人民币 1,240.50 亿元，较上年度实际执行总额增长 8.4%。

二、 产品分项额度建议

产品种类 拟授信额度(亿元) 占比
传统融资担保 850.00 68.5%
科技型融资担保 220.50 17.8%
“三农”专项担保 170.00 13.7%

三、 风险管控及执行要点

严格落实“压降、控额、提质”的总体原则。针对风险等级评定为 C 级及以下的机构，其新增业务规模必须严格控制在上一年度结余的 90% 以内。
